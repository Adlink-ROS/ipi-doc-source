## ADLINK IPi documentation


### Getting started



1. Install dependencies:


```
$ git clone https://github.com/ADLINK-EPM/ipi-doc-source.git
$ cd ipi-doc-source
$ npm install
$ npm install hexo-cli -g

```

2. Generate HTML website:

```
$ hexo clean
$ hexo g
```

3. Run server on local side:

```
$ hexo s
```
